Floyd Warshall Shortest Path

Programming Language: c++

Project Type: Algorithm

Link to Github Page

This project is a shortest path algorithm in c++ using Floyd Warshall algorithm. This program allows users to create nodes and add weights to the path between each of the nodes. After users input a start and end node, it will calculate the shortest path between these two nodes and output the path and the total weight of the path.



Template Double Ended Queue

Programming Language: c++

Project Type: Data Structure

Link to Github Page

This project is a dynamic template double ended queue in c++. It allow users to create the double ended queue data structure of any data type (i.e. String, int, or any custom object type) as long as there is a toString function for that object. This double ended queue also grows dynamically with the number of items currently added to the queue. Its initial size is 8 but will increase or decrease depending on how many are added or removed.



OpenGL Points and Curves

Programming Language: c++

Project Type: OpenGL Graphics Project

Link to Github Page

An OpenGL c++ project that allows user to click to create new points on the screen. After ten points are added the program uses De Casteljau algorithm to calculate the De Casteljau smooth curves and draws them. This program uses a vector (not an array) to store the points thus not limiting the number of points that can be created. These points store a vertex and point structs to more easily manipulate.



VR Da Vinci's Workshop Simulation

Programming Language: C# and Jave Script

Project Type: VR Unity Project

Link to Project Website

In this VR game created in Unity with C# and javascript, the user can visit a simulation of Leonardo Da Vinci's Workshop and see the amazing machines Da Vinci created come to life. There are two different types of interactions, first an AR like experience where the user can see two inventions come to life on a virtual table and the second where the user can operate Da Vinci's invention and experience it in first person.
